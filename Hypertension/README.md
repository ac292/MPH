# Hypertension Project Analysis and Simulation
We look at the impact of hypertension in men and women as they age based on CDC data.
* Does hypertension affect men and women in the same fashion? 
* Has there been a change in the impact of hypertensive disease between the years 1990s and the late 2010s?

## Usage
Open the R markdown file Hypertension.Rmd in RStudio and click "Knit"

## Folder Structure
- analysis/ -R Markdown scripts
- data/ -datasets
- output/ - generated pdf and plots
